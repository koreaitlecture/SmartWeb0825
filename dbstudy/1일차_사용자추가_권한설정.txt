모두 관리자로 접속한 뒤 처리해야 함.
conn system/123456

1. 사용자 추가
create user 사용자 identified by 비밀번호;

2. 권한 부여
grant connect, resource to 사용자;  -- 접속 및 자원 사용 권한
grant dba to 사용자;  -- 관리자 권한

3. 사용자 제거
drop user 사용자;
drop user 사용자 cascade;  -- 사용자가 테이블 등 데이터를 가진 경우

4. 권한 해제
revoke connect, resource from 사용자;
revoke dba from 사용자;
